# 📈 Harini's Live Market Dashboard

An interactive stock market dashboard built using **Python**, **Streamlit**, **Plotly**, and **yFinance** for real-time analysis of NIFTY 50 stocks.

## Features
- 📊 Live NIFTY 50 Dashboard
- 📈 Interactive Line & Candlestick Charts
- 🚀 Volume Speedometer
- 📋 Professional Market Scanner
- 🔍 Stock Search
- 📉 EMA (20, 50, 200)
- 📊 RSI Indicator
- 📈 MACD Indicator
- 🏦 Sector Performance
- 🚀 Breakout Scanner
- ⚡ Momentum Scanner
- 📊 Market Breadth Analysis
- 🔥 High Volume Stocks
- ⭐ Stocks Near 52-Week High/Low

## Technologies Used
- Python
- Streamlit
- Plotly
- Pandas
- yFinance

## Data Source
Market data is retrieved in near real time using the yFinance library, which sources publicly available Yahoo Finance market data.
